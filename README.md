# ⏰ Analog + Digital Clock

A **responsive Analog + Digital Clock** built using **HTML, CSS, and JavaScript**.  
This project displays the **current time in both analog and digital formats** with smooth animations and real-time updates.

It is a **simple yet powerful frontend project** for beginners to learn **JavaScript DOM manipulation, CSS animations, and real-time web applications**.

---

## 🚀 Features

- 🕒 Real-time Analog Clock
- ⌚ Digital Time Display
- 🎨 Clean and modern UI
- ⚡ Smooth clock hand animation
- 📱 Fully responsive design
- 🔄 Automatic time update every second

---

## 🛠️ Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**

## 🎯 How It Works

JavaScript uses the **Date() object** to get the current system time and updates:

- Analog clock hands rotation
- Digital clock display

The clock updates **every second using `setInterval()`**.
